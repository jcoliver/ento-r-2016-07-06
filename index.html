Preparation

You will need to install two pieces of software on your laptop for this workshop, R and RStudio. Installation for R may vary by your operating system; follow the instructions below for your OS. If you have questions about installation, or run into problems, please feel free to e-mail me at jcoliver@email.arizona.edu.

R

Windows

Install R by downloading and running the exe file: R Windows Installer.

Mac OSX, release 10.6 and above

Install R by downloading and running the pkg file: Mac OSX package.

Linux

Download the binary files for your Linux distribution from CRAN. Alternatively, you can use your package manager to install R (e.g. for Debian/Ubuntu, run sudo apt-get install r-base and for Fedora run sudo yum install R).

RStudio

Download and install RStudio

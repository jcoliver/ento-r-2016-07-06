Introduction

This workshop is designed for a novice audience and requires no prior knowledge of R or other programming languages. We will begin with an introduction to the language and the RStudio interface and tentatively end with creating publication- quality graphics. The learning objectives are:

  1. To understand how information is stored in R
  2. To learn how to get data into and out of R
  3. To apply DRY principles to programming (DRY = Don't Repeat Yourself)
  4. To create high-quality data visualizations

Preparation

You will need to install two pieces of software on your laptop before this workshop: R and RStudio. Installation for R may vary by your operating system; follow the instructions below for your OS. If you have questions about installation, or run into problems, please feel free to e-mail me at jcoliver@email.arizona.edu.

R

Windows

Install R by downloading and running the exe file: R Windows Installer.

Mac

For Mac OSX, release 10.6 and above, install R by downloading and running the pkg file: Mac OSX package.

For earlier releases of Mac OS, see details on installing older versions of R.

Linux

Download the binary files for your Linux distribution from CRAN. Alternatively, you can use your package manager to install R (e.g. for Debian/Ubuntu, run sudo apt-get install r-base and for Fedora run sudo yum install R).

RStudio

Download and install RStudio
